    VISIT SITE
    • 기획방향 - Planning direction
      기여도 100%

      전 세계 다이빙 명소를 소개하는 웹사이트로서, 사용자에게 각 명소의 특색을 시각적으로 강조하며 몰입도 높은 경험을 제공하는 것을 목표로 하였습니다. HTML5 표준을 준수하고, 반응형 웹사이트로 제작하여 다양한 기기에서 최적화된 사용자 경험을 제공하였습니다. 또한, 다이빙이라는 주제에 맞게 시각적 요소를 통해 명소의 분위기를 생생하게 전달하는 데 중점을 두었습니다.

    • 제작전략 - Coding Production strategy
      기여도 100%

      Media Query를 활용해 와이드 PC(1281px~), 일반 PC(1025px1280px), 태블릿(641px1024px), 모바일(~640px)까지 폭넓은 해상도에서 유연한 레이아웃을 구현했습니다. Flex와 Grid를 사용해 다양한 레이아웃을 적용하고, CSS Nesting 기법을 활용하여 코드의 간결성과 재사용성을 높였습니다. 스크롤 이벤트를 적용해 다이버가 나타나고, 다이버의 장비에서 공기방울이 나오는 애니메이션을 통해 웹사이트에 생동감을 더하며, 다이빙이라는 주제에 맞는 시각적 경험을 강화했습니다. 또한, 모바일 화면에서는 햄버거 메뉴를 구연하였고 메뉴에 귀여운 아이콘을 적용해 시각적 즐거움을 더했습니다.

      HTML5 / CSS3 / JavaScript / jQuery / AJAX / 그누보드(게시판) / W3C 웹표준 통합 마크업 검사(html, css) / 웹접근성검사(K-WAH) / 크로스브라우징 완료

    • 디자인 - Design
      기여도 100%

      Flex와 Grid 레이아웃을 사용하여 명소 소개 페이지에서 시각적 일관성을 유지하였으며, 사용자 중심의 직관적 네비게이션을 통해 다이빙 명소 탐색을 쉽게 할 수 있도록 했습니다. 다양한 기기에서 최적화된 인터페이스를 제공하며, 모바일에서는 햄버거 메뉴를 사용해 사용자 편의성을 극대화했습니다. Masonry 플러그인을 사용해 갤러리 페이지를 구성하여 다양한 명소 이미지를 효과적으로 배치하고 시각적 매력을 더했습니다.
